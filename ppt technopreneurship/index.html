02 — Permasalahan
Identifikasi Permasalahan
1

Proses Manual Memakan Waktu

Absensi kertas dilakukan berurutan satu per satu. Untuk 30 peserta, waktu terbuang ±5 menit per sesi dan mengganggu waktu pembelajaran.

2

Rentan Kecurangan

Titip absensi, pemalsuan tanda tangan, dan pengisian oleh pihak yang tidak berhak sulit terdeteksi, mengurangi validitas data kedisiplinan.

3

Rekapitulasi Rawan Error

Data kertas dikumpulkan, disortir, lalu diinput manual ke digital. Proses repetitif ini sangat rentan human error dan ketidaksesuaian data.

4

Keterbatasan Akses Data

Data hanya tersedia fisik di lokasi. Pihak berkepentingan tidak bisa memantau kehadiran real-time dari jarak jauh melalui platform digital.

04 — Teknologi
Teknologi yang Digunakan

ESP32

Espressif Systems

• Dual-core 32-bit 240MHz
• Built-in Wi-Fi & Bluetooth
• Otak pemroses & pengirim data

NFC PN532

13.56 MHz

• Baca UID kartu/tag NFC
• Jarak baca 3–5 cm
• Protokol I2C, baca < 1 detik

Web Stack

PHP + MySQL

• PHP + CodeIgniter
• MySQL 8.x via XAMPP
• Dashboard responsif

LCD I2C 20x4 Kartu NFC (UID) Arduino IDE 2.x (C/C++) HTTP POST JSON

Arsitektur: Input (NFC) → Proses (ESP32) → Komunikasi (Wi-Fi) → Penyimpanan (MySQL) → Antarmuka (Web)

07 — Inovasi
Aspek Inovasi

Integrasi IoT End-to-End

Menggabungkan hardware (ESP32+NFC), komunikasi Wi-Fi/HTTP, dan web (PHP+MySQL) dalam satu ekosistem presensi utuh.

Zero-App Approach

Tidak perlu instalasi aplikasi. Semua manajemen dan monitoring melalui browser, menghilangkan barrier adopsi teknologi.

Low-Cost High-Impact

Membuktikan sistem presensi digital terintegrasi bisa dibangun < Rp 400rb dengan komponen open-source, replikable untuk institusi beranggaran terbatas.

Notifikasi WhatsApp Otomatis

Data kehadiran diteruskan otomatis ke wali murid via WhatsApp, meningkatkan transparansi dan komunikasi sekolah-orang tua.

08 — Hasil Diharapkan
Hasil yang Diharapkan

Prototype Fungsional

Sistem presensi otomatis lengkap: hardware (ESP32+NFC) + software web yang mencatat data dari tap kartu hingga tampil di dashboard.

Efisiensi Proses

30 peserta: dari 5–10 menit (manual) menjadi < 1 menit. Waktu efektif pembelajaran bertambah signifikan.

Akurasi & Transparansi

UID unik menghilangkan titip absensi. Data otomatis tersimpan di database, bisa dipantau real-time dosen dan orang tua.

Bukti Penerapan IoT

Pengalaman langsung menerapkan IoT untuk masalah nyata + dokumentasi terstruktur sebagai referensi pengembangan.
